# Desktop Notifier & Task Manager

This is a versatile Python-based utility that enables users to schedule custom desktop alerts with automated asset generation to ensure a seamless user experience.

⚠️ ""Developed as a practical exercise to showcase system-level API integration and event-based scheduling in Python.""

### Setup Guide

1. Verify that Python 3.8+ is installed on your machine.
2. Launch your preferred Terminal or IDE.
3. Setup dependencies via pip: `pip install plyer Pillow requests`
4. Execute the application: `python new_project.py`

### Project Goal

The primary objective of this tool is to master local environment interactions and background process management. It addresses typical development challenges such as dynamic icon creation and persistent loop handling without high CPU usage.

### Core Capabilities

-> **Reliable Alerts:** Utilizes native OS protocols to ensure high-priority visibility.

-> **Dynamic Assets:** Automatically builds visual components if local files are missing.

-> **Error Resilience:** Features integrated try-except blocks to catch and log input anomalies.
